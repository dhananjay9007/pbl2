# Streamlit Survey Analytics Dashboard

This repository contains a Streamlit app that provides:
- Classification (Random Forest)
- Association Rule Mining (mlxtend apriori & association_rules)
- Clustering (KMeans)
- Regression (Random Forest Regressor)
- Interactive filters in the sidebar applied only to the dashboard view

## How to run locally
1. Install requirements: `pip install -r requirements.txt`
2. Place your dataset `survey_data.csv.csv` into the same folder as `app.py`.
3. Run: `streamlit run app.py`
